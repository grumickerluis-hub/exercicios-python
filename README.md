# 🐍 Exercícios em Python

Repositório criado para registrar minha evolução no estudo da linguagem Python durante a graduação em Ciência da Computação.

## Objetivo

Praticar lógica de programação, estruturas de decisão, estruturas de repetição, funções e organização de código utilizando Python.

## Conteúdos praticados

- Entrada e saída de dados
- Variáveis e operadores
- Estruturas condicionais
- Laços de repetição
- Funções
- Contadores e acumuladores
- Cálculo de médias
- Validação de resultados
- Formatação de textos

## Exercícios

- Verificação de número par ou ímpar
- Cálculo da média de alunos
- Classificação de aprovação, recuperação e reprovação
- Tabuada
- Soma de valores
- Contagem de números pares
- Sequência de Fibonacci
- Exercícios com funções

## Tecnologias utilizadas

- Python
- VS Code
